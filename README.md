# Agent Patterns

Reference implementations and patterns for building tool-using agents with the Anthropic Python SDK.

## Contents

| File | What it covers |
|---|---|
| `agent_loop.py` | Canonical agent loop with tool dispatch and debug prints |
| `tool_spec_reference.md` | Tool definition schema, tool_result format, response object attributes |
| `debug_methods.md` | Four debug techniques + symptom-to-method map |
| `optimization_playbook.md` | System prompt skeleton, tool description improvements, tool redesign patterns |
| `python_quick_ref.md` | Python patterns commonly used in agent code |
| `error_reference.md` | Common error messages, what they mean, and how to fix them |
| `checklist.md` | Pre-run sanity check for agent code |
